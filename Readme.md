# Paediatric Malaria Triage Agent

An AI-powered conversational triage assistant for community health 
workers (CHWs) assessing febrile children under 5 years in 
low-income countries.

## What it does
- Classifies fever severity by age and temperature
- Screens for WHO IMCI danger signs
- Assesses malaria risk by geographic region
- Differentiates malaria, typhoid, meningitis, and viral fever
- Recommends home care, clinic referral, or emergency evacuation
- Retrieves real-time evidence from PubMed and Wikipedia

## Built with
- Anthropic Claude Sonnet 4.6
- LangChain + LangGraph
- FastAPI
- Deployed on Render

## Live API
https://malaria-triage.onrender.com

## Clinical basis
WHO Integrated Management of Childhood Illness (IMCI) guidelines
Pubmed and wikipidia evidence based publications

## Target users
Community health workers in sub-Saharan Africa
